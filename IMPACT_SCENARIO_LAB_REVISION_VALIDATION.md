# Impact Scenario Lab Revision Validation

This validation confirms the requested Impact Scenario Lab changes were implemented.

## Implemented changes

- Graph line labels were changed from bold to lighter-weight labels for readability.
- A mode-toggle button was added to switch between Relative Impact and Financial Impact.
- Relative Impact mode uses `I(t)=C(t)/P`.
- Financial Impact mode plots flat cumulative dollar loss values directly.
- The exact scenario figures panel was moved above the graph.
- The explanatory "what the animation shows" panel was moved above the graph.
- Formal PhD notation was added to the page.
- The page explicitly states that the visualisation backs up Mark Dorsett's PhD claim that denial impact is victim-asymmetric.
- Live cards update in both relative-impact and financial-impact modes.
- Axis labels and tick labels update when the graph mode changes.
