D-ACT Browser Classifier

A static, client-side web interface for the C0-C6 Denial Attack Classification Tool. Upload PCAP, flow, cloud, serverless and AI usage artefacts, then inspect the inferred conditions and taxonomy classification.

Client-side only No server required PCAP / CSV / JSON / XLSX C0-C6 taxonomy visualisation

1. Upload evidence files

Files are processed locally in the browser. They are not uploaded to a remote server by this static implementation.

Use classic Ethernet/IPv4 .pcap files. If your capture is .pcapng, convert it with editcap -F pcap input.pcapng output.pcap before use.
Ready.

      Repository notes

      This website is designed for static hosting. Upload index.html, assets/, and optionally examples/ to a public repository. The classifier runs in the browser and mirrors the D-ACT proof-of-concept thresholds used in the paper.

      PCAP-only analysis can infer C0, C1, C2, and C3. Cloud logs are required for C4, serverless logs for C5, and AI usage logs for C6.
