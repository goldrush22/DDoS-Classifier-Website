A guided, non-technical catalogue of 19 synthetic D-ACT examples. Each example includes the evidence files, expected conditions, downloadable artefacts, a browser preview, and a one-click in-browser run.

How to use these examples

Each card below represents one synthetic offline use case. Select Run example to execute the bundled files through the browser version of D-ACT. Select Preview beside any file to inspect it before downloading. PCAP previews show parsed packet metadata rather than raw binary content.

These examples are synthetic reproducibility artefacts. They are not live attack traces and do not generate network traffic.
